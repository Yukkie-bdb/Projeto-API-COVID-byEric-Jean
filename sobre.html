PROJETO API COVID-19
Painel De Casos Sobre o Projeto Equipe de DEV's
SOBRE O PROJETO
O projeto tem como objetivo a criação propria de um API para integrar dados do site https://covid19-brazil-api-docs.vercel.app/ para um site criado pelos os alunos do curso de Desenvolvimento de Sistemas do SENAI.
NOSSO OBJETIVO
Temos como o objetivo criar um Site responsivo usando as linguagens HTML, CSS e JAVASCRIPT que pegue informações de um site de terceiros e as integreas ao nosso Site usando uma API criada por nois mesmos.
FERRAMENTAS E TECNOLOGIAS USADAS
Usamos diversas ferramentas para a criação desse projeto, sendo para a edição o programa VISUAL STUDIO CODE, usamos as linguagens HTML, CSS e JAVASCRIPT, onde no JAVASCRIPT criamos a nossa propri API para a integração de dados de fora, o CSS para a estilização do Site e HTML para programar a sua base.
CREDITOS
Todos os Dados utilizados nesse projeto foram retirados do site https://covid19-brazil-api-docs.vercel.app/. Usamos tambem os graficos do Google Charts, sendo eles os Grafico Mapa, Grafico de Pizza e tambem uma Tabela.
